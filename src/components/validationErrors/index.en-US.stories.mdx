# ValidationErrors

## Description

A component for displaying error messages

## When To Use

- When you want to display application error messages as a result of `HTTP` request or any messages.

## API

### Initialization/Props

| Property | Description                  | Type                                             | Default | Required |
| -------- | ---------------------------- | ------------------------------------------------ | ------- | -------- |
| error    | the error message to display | IAjaxResponse \| IErrorInfo \| string \| boolean |         | true     |
